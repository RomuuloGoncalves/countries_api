# Country Search Project
This is a simple web application that uses the REST Countries API to allow users to search for countries and view information about them, such as name, flag, population, capital, currency, language, and more.

The application is built with HTML, CSS, Bootstrap, and JavaScript.

## Features
* Users can search for a country by name, capital, or country code.
* The list of corresponding countries is displayed in real time as the user types.
* Clicking on a country in the list displays detailed information about the country, including its flag, name, capital, population, currency, language, and more.
* Users can mark a country as a favorite, and the application saves this information in the browser's local storage.
* Users can view the list of favorite countries at any time.
